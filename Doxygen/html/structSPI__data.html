Detailed Description

This structure contains all info necessary to communicate with and to interpret incoming data from the Honeywell HSC sensors. Please refer to the datasheet and Honeywell SPI companion for info on parameters like P__OUT_MAX, P_OUT__MIN, etc. Note that the specific pressure sensors that we use are: HSC D LN N 100MD S A 5.

void startVideo ( char *  filename,
char *  options 
)

This function starts the Raspberry Pi camera recording with specific recording options *options stored at the adress pointed to by options. If you want to enable preview/monitoring then make the obvious change to remove the -n (no preview) option from the source code of this function.

Parameters
filenameA string specifying the name of the resulting video file (must have .h264 ending).
optionsNormal raspivid options. Avoid -t, -n, -o, and -s as the code fills those in for you.
